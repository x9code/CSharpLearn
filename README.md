# CSharpLearn

A structured C# learning repository covering key concepts from dependency injection and exception handling to LINQ, structs, and debug logging.

## 📁 Project Structure

```
CSharpLearn/
├── DebugLogs/              # Debugging and logging examples
├── DependencyInjection/    # DI patterns and usage in C#
├── ExceptionPrograms/      # Exception handling techniques
├── Linq/                   # LINQ queries and operations
├── StructsApp/             # Working with structs in C#
├── CSharpLearn.csproj      # Project configuration file
├── CSharpLearn.sln         # Solution file
├── .gitattributes
└── .gitignore
```

## 📚 Topics Covered

### 🔧 Dependency Injection
Demonstrates how to use the built-in .NET dependency injection system, including service registration, constructor injection, and interface-based design.

### ⚠️ Exception Programs
Covers exception handling in C# including:
- `try-catch-finally` blocks
- Custom exception classes
- Best practices for error handling

### 🔍 LINQ
Explores Language-Integrated Query (LINQ) features:
- Filtering, sorting, and projecting collections
- Method syntax vs query syntax
- Working with `IEnumerable<T>`

### 🏗️ Structs App
Demonstrates the use of value types (structs) in C#:
- Defining and using structs
- Structs vs classes
- Immutability and performance considerations

### 📝 Debug Logs
Examples related to logging and debugging:
- Using `Debug` and `Trace` classes
- Structured logging patterns
- Debugging techniques in .NET

## 🚀 Getting Started

### Prerequisites
- [.NET SDK](https://dotnet.microsoft.com/download) (6.0 or later recommended)
- Visual Studio / Visual Studio Code / Rider

### Running the Project

1. **Clone the repository**
   ```bash
   git clone https://github.com/x9code/CSharpLearn.git
   cd CSharpLearn
   ```

2. **Build the solution**
   ```bash
   dotnet build CSharpLearn.sln
   ```

3. **Run the project**
   ```bash
   dotnet run --project CSharpLearn/CSharpLearn.csproj
   ```
